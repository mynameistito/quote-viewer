# Code Of Conduct

## Our Standards

This project should be a respectful, focused place to improve `quote-viewer`.

Examples of welcome behavior include:

- Being kind, patient, and constructive in discussions.
- Giving clear, actionable feedback on issues and pull requests.
- Respecting different levels of experience and different browser/platform setups.
- Staying on topic and assuming good intent when something is unclear.

Examples of unacceptable behavior include:

- Harassment, insults, threats, or personal attacks.
- Discriminatory language or behavior.
- Sexualized language or imagery.
- Repeatedly derailing issues or pull requests.
- Publishing private information without permission.

## Enforcement

Maintainers may edit, hide, lock, or delete comments, issues, pull requests, and other contributions that violate this Code of Conduct. Maintainers may also temporarily or permanently restrict participation when needed to protect the project and its contributors.

If you see behavior that violates this policy and it is safe to discuss publicly, open an issue at <https://github.com/mynameistito/quote-viewer/issues/new/choose>. If the report is sensitive, involves private information, or involves a maintainer, use GitHub's platform reporting tools at <https://github.com/contact/report-abuse> instead.

## Scope

This Code of Conduct applies in this repository and in project-related spaces, including issues, pull requests, discussions, reviews, and release communication.
